# BlindCommand 使用说明

> 盲棋指挥 — 以信息不对称为核心玩法的实时战术指挥游戏

---

## 1. 环境要求

| 项 | 要求 |
|---|------|
| 操作系统 | Windows / macOS / Linux |
| Python | **3.11**（其他版本未测试） |
| 内存 | ≥ 512 MB |
| 显卡 | 不需要独立显卡，pygame 纯 CPU 渲染 |

---

## 2. 安装与运行

### 2.1 一行运行（Windows 用户）

双击项目根目录下的 `run.bat`，脚本会自动：
1. 检测 Python 是否安装
2. 激活虚拟环境（如果存在 `.venv`）
3. 安装依赖 `requirements.txt`
4. 启动游戏

### 2.2 手动安装运行

```bash
# 1. 克隆项目
git clone <repo-url>
cd BlindCommand

# 2. （推荐）创建虚拟环境
python -m venv .venv
# Windows:
.venv\Scripts\activate
# macOS/Linux:
source .venv/bin/activate

# 3. 安装依赖
pip install -r requirements.txt

# 4. （可选）安装噪声库（地图生成需要）
pip install noise cairosvg

# 5. 运行
python -m src.main
```

### 2.3 开发环境

```bash
# 安装开发依赖（格式化/类型检查/测试）
pip install -r requirements-dev.txt

# 安装 pre-commit 钩子
pre-commit install

# 运行测试
pytest tests/ -v
```

## 3. 游戏流程

```
┌──────────┐    ┌──────────────┐    ┌──────────────────────────────────┐
│  关卡选择  │ → │  观察开局战报  │ → │  主循环（60fps 实时）              │
│  简单/中等  │    │  了解初始态势  │    │                                │
│  困难      │    │              │    │  点目标 → 选单位 → Y确认/N取消    │
└──────────┘    └──────────────┘    │       ↓                         │
                                    │  指令延迟 ~1秒                    │
                                    │       ↓                         │
                                    │  部队移动/自动交战                │
                                    │       ↓                         │
                                    │  收到战报 + 汇报圈（短暂闪烁）     │
                                    │       ↓                         │
                                    │  根据战报在图上标记推测位置        │
                                    │       ↓                         │
                                    │  循环直到胜利/失败 → 结算画面      │
                                    └──────────────────────────────────┘
```

### 胜利条件
- **胜利**：消灭所有敌军 或 占领敌方 HQ
- **失败**：己方全军覆没 或 己方 HQ 被占领

## 4. 操作详解

### 4.1 指挥部队

由于地图上看不到任何单位，你只能根据**左侧战报**中描述的位置来推测部队在哪里。

**发出移动指令：**

| 步骤 | 操作 | 说明 |
|------|------|------|
| ① 读战报 | 查看左侧面板 | 战报会用**地形参照**描述位置，如"第一步兵连在森林以东、河流以南约2格" |
| ② 点目标 | 在地图上点击目标位置 | 地图为右侧区域（泛黄亚麻纸纹理 + 地形符号） |
| ③ 选单位 | 弹出的浮动面板中选择 | 可选单位以连队名称列出（如"1连""2连""骑1"） |
| ④ 确认 | 按 **Y** 确认 / 按 **N** 取消 | Y=执行，N=放弃 |

指令发出后有约 **1秒延迟** 才会到达部队，模拟通讯延迟。

### 4.2 标记推测

标记是你记录推测单位位置的工具——**标记不影响游戏实际状态**，只存在于你的地图上。

| 操作 | 方式 |
|------|------|
| **放置标记** | 从左侧**调色板**拖拽数字（1-10）或兵种图标到地图上 |
| **移动标记** | 拖拽已放置的标记到新位置 |
| **删除标记** | 右键点击地图上的标记 |
| **撤销删除** | 按 Z 键（在选中标记后） |

**调色板位置**：战报面板下方，包含：
- 数字标签 1-10：用于标记号部队的推测位置
- 兵种图标：步兵/骑兵/炮兵/侦察兵/HQ × 蓝色(友军) + 红色(敌军)

### 4.3 快捷键

| 按键 | 功能 |
|------|------|
| **鼠标左键点击地图** | 选择目标位置 / 放置标记 |
| **鼠标右键点击标记** | 删除标记 |
| **鼠标滚轮（左侧区域）** | 滚动战报 |
| **Y** | 确认移动指令 |
| **N** | 取消移动指令 |
| **空格 / P** | 暂停 / 恢复游戏 |
| **ESC** | 退出游戏（DEV 模式下需按两次，第一次退出 DEV 模式） |

### 4.4 开发者模式

开发者模式让你看到**所有单位的真实位置**（蓝色=友军，红色=敌军），用于调试或学习。

1. 点击左上角 **DEV** 按钮
2. 输入密码 `123456` → 回车
3. 地图上会显示所有单位位置
4. 再次点击 DEV 按钮 或按 ESC 退出开发者模式

## 5. 游戏机制

### 5.1 信息不对称（核心特色）

> **你看不到任何单位的真实位置。永远看不到。**

你获取战场信息的唯一途径：

| 信息源 | 内容 | 频率 |
|--------|------|------|
| **战报文字** | 地形参照描述（"骑兵一连在桥梁附近与敌军遭遇"） | 实时推送 |
| **汇报圈** | 地图上短暂闪烁的圆形区域，指示己方单位大致位置（误差 ±1 格） | 每 3-5 秒 |
| **标记** | 你自己在地图上做的推测记录 | 手动操作 |

**设计原则**：战报中**不出现坐标数字**，只使用"在森林以东""距河流约2格"等地形参照语言。

### 5.2 战斗系统

- **自动遭遇**：双方单位相邻（8 方向）时自动触发交战
- **交战延迟**：首次接触后有 ~1 秒准备时间
- **战斗轮**：每轮 2 秒，双方交替攻击
- **兵种克制**：步兵 克 骑兵 克 炮兵 克 步兵
  - 克制方伤害 ×2.0
  - 同等兵种伤害 ×1.0
  - 被克方伤害 ×0.5
- **撤退规则**：HP 低于 20% 时可能溃逃
- **地形加成**：不同地形提供防御加成（森林 +30%、山地 +50%）

### 5.3 移动速度

| 兵种 | 每格耗时 | 说明 |
|------|---------|------|
| 骑兵 | 1.0 秒 | 最快 |
| 侦察兵 | 1.2 秒 | 快，视野大 |
| 步兵 | 2.0 秒 | 中等 |
| 炮兵 | 6.0 秒 | 最慢，火力强 |
| HQ | 不可移动 | 指挥部固定位置 |

### 5.4 AI 难度

| 难度 | 特点 |
|------|------|
| **简单** | 被动防守，仅在小范围巡逻 |
| **中等** | 主动索敌，有基本战术判断 |
| **困难** | 集中兵力、包抄侧翼、优先攻击弱点 |

当前版本暂不支持命令行参数，所有配置通过关卡选择界面完成。未来版本将支持：

## 8. 常见问题

### Q: 启动时报 `IndentationError` 或其他语法错误？
A: 确认使用 Python 3.11，其他版本可能存在兼容性问题。

### Q: 地图是黑屏/空白？
A: 检查 `data/maps/map_01.json` 是否存在。首次运行会自动生成。如果仍然空白，尝试安装 `noise` 库：`pip install noise`。

### Q: 中文显示方框/乱码？
A: 项目自带中文字体 (`data/chinese.ttf`)。如果显示异常，请确认该文件存在，或安装系统字体如"微软雅黑""黑体"。

### Q: 战报不滚动？
A: 鼠标滚轮只在**左侧战报面板区域**（窗口左 28%）生效。

### Q: 如何打包成 exe？
```bash
pip install pyinstaller
pyinstaller --onefile --add-data "data;data" --add-data "src;src" src/main.py
```

### Q: 如何快速学习游戏？
A: 建议先打开**开发者模式**（密码 `123456`），观察单位如何移动和战斗，然后再关闭 DEV 模式体验真正的盲棋。
