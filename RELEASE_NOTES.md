## 🎯 概述

v0.3.0 是 BlindCommand 从**回合制原型**迈向**实时战术（RTT）游戏**的关键里程碑。核心重构了游戏引擎、战斗系统、地图生成和 UI 交互，实现了真正意义上的"盲棋指挥"——指挥官只能通过战报文字和短暂汇报圈感知战场，永远看不到单位的真实位置。

---

## 🔥 重大变更

### ⚡ RTT 实时引擎

- **从回合制到实时**：`GameLoop.run_turn()` 替换为 `RealTimeEngine.update(dt)`，采用连续时间步进（60fps）
- **浮点坐标移动**：单位以浮点坐标平滑移动，不再按格子跳转
- **速度差异化**：骑兵 1.0s/格、侦察兵 1.2s/格、步兵 2.0s/格、炮兵 6.0s/格
- **指令延迟**：发出指令后约 1 秒延迟到达，模拟一战通讯延迟
- **自动遭遇战**：双方单位相邻时自动触发，1 秒接敌延迟 + 每轮 2 秒战斗

### 🗺️ Perlin 噪声地图生成

- 基于 Perlin/Simplex 噪声生成自然地形（平原 / 森林 / 山地）
- 河流模板叠加，1-3 格宽度
- BFS/DFS 连通性保证（最多 100 次重试，失败后自动加桥或移除孤立山地）
- 每局生成不同地图，支持 10×10 到 40×40 尺寸

### 🎨 UI 全面改造

- **真盲棋显示**：地图上**永不渲染单位**，仅显示地形 + 玩家标记 + 短暂汇报圈
- **标记系统升级**：支持拖拽数字 1-10 + 兵种图标到地图，右键删除，拖拽移动，Z 键撤销
- **浮动指令面板**：点击地图弹出，自动计算方向/距离，Y 确认 / N 取消
- **战报叙事化**：位置描述使用地形参照语言（"在森林以东、距河约 2 格"），不出坐标数字
- **汇报圈**：地图上短暂闪烁的圆形区域，指示己方单位大致位置（±1 格误差）
- **开发者模式**：密码 `123456` 进入，显示所有单位实时位置（蓝=友军、红=敌军）
- **关卡选择 + 结算画面**：支持简单/中等/困难三级难度，结算显示击杀数、存活数、用时

### 🤖 AI 增强

- **三级难度**：简单（被动防守）、中等（主动索敌）、困难（集中兵力/包抄侧翼）
- **行为树决策**：撤退（HP<20%）> 战斗 > HQ 防御 > 巡逻
- **兵种差异化**：各兵种不同巡逻范围、接敌策略

## 🐛 Bug 修复

### 核心层 (Core)
- 修复地形防御残留导致同一地块多次累计防御加成
- 修复嵌套 `emit()` 导致事件总线死锁
- 修复 A* 寻路中 dead-code 路径永不执行
- 修复迷雾系统边界检查缺失导致坐标越界
- 修复 `RangeQuery` Chebyshev 距离计算符号错误

### 战斗层 (Battle)
- 修复 `Commander` 指令队列时间戳漂移
- 修复战斗结算后单位 HP 未正确 clamp 到 [0, max]
- 修复撤退逻辑在河流/山脉地形判断错误
- 修复 AI 在无可用目标时陷入死循环

### UI 层 (UI)
- 修复 `main_window.py` 缩进错误导致 `UI_TEXT_ENTRY_CHANGED` 处理器无法运行
- 修复 `MarkerSystem` 四个缺失属性（`_hovered_type`、`_palette_dirty`、`_last_deleted`、`_font`）
- 修复 `MainWindow._layout` 未初始化导致装饰绘制崩溃
- 修复 PyInstaller onefile 模式下图片/地图/字体路径错误
- 修复 Windows 下中文字体显示为白色方块
- 修复主题文件字体路径不生效

### 代码质量
- 删除所有废弃 `_old` 文件
- 清理未使用的导入和资源文件
- 全面通过 ruff / isort / black 检查
- 修复 `undefined constants` 和 `SIM102` 规则违规

## 📦 安装与运行

```bash
pip install -r requirements.txt
pip install noise cairosvg    # 地图生成所需（可选）
python -m src.main
```

或双击 `run.bat`（Windows 一键启动）。

详见 [USAGE.md](USAGE.md)。
